# Web agency site build with expressjs

## Getting started

1. Install dependencies

  ```bash
  npm install
  ```

2. Run

  ```bash
  npm start
  ```

## Dockerized version

1. Build docker image

  ```bash
  docker build -t agency .
  ```

2. Run docker container

  ```bash
  docker run -d -p 8000:3000 agency
  ```

3. Navigate to `localhost:8000`
